# Calculator_Project_Shuranov_Egor
## О проекте

**Calculus Tool** - это консольное приложение с расширенными функциями. Нужен мощный калькулятор прямо в консоли? Этот инструмент для вас! Он умеет не только складывать и вычитать, но и выполнять более сложные математические операции и много другого. Идеально подходит для всех, кто имеет дело с цифрами.

## Функциональность

### Математические операции
- **Базовые операции**: сложение, вычитание, умножение, деление
- **Расширенные операции**: возведение в степень, вычисление квадратного и кубического корня
- **Деление с остатком**: вычисление остатка от деления
- **Сравнение чисел**: проверка больше, меньше, равно
- **Проценты**: вычисление процентов от числа

### Дополнительные функции
- **Конвертер валют**: расчет суммы по актуальному курсу
- **История вычислений**: просмотр последних операций
- **Обработка ошибок**: корректная обработка деления на ноль и неверного ввода

## Требования

- **Python**: версия 3.8 или выше
- **Библиотеки**: стандартные библиотеки Python (внешние зависимости не требуются)
